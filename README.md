# Graham Zemel (.com)

[![Vercel](https://img.shields.io/badge/vercel-deployed-000?logo=vercel&style=flat-square)](https://grahamzemel.com)

Personal portfolio and admin dashboard for [Graham Zemel](https://grahamzemel.com).

- **Live site:** [https://grahamzemel.com](https://grahamzemel.com)
- **Stack:** SvelteKit, Tailwind CSS, Vite, TypeScript
- **Hosted on:** Vercel
- **Backend:** [grahamzemel.com_backend](https://github.com/grahamzemel/grahamzemel.com_backend)

## About

This is the fourth iteration of my personal site — a single-page portfolio built with SvelteKit and Tailwind CSS. It showcases projects, writing, work experience, and a contact form, with a password-protected dashboard on `/admin`.

## Sections

- **Hero** — intro, tagline, and social links
- **About** — bio, current focus, and tech stack
- **Featured Projects** — FratDoor, TextCloaker, offline-sync-engine
- **Projects** — searchable, filterable grid of 20+ projects
- **Resume** — embedded résumé with download
- **Blog** — latest posts from The Gray Area
- **Get in Touch** — contact form

## Tech stack

- [SvelteKit](https://kit.svelte.dev)
- [Vite](https://vitejs.dev)
- [Tailwind CSS](https://tailwindcss.com)
- [GSAP](https://greensock.com/gsap) + [Lenis](https://lenis.studiofreight.com)
- [TypeScript](https://www.typescriptlang.org)
